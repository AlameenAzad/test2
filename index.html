Shack Shack Chatbot

I have had the experience to help develop a bot with 60,000+ users. I had multiple aims in this project. The owner used to go through manual image editing per request from the bot. Therefore I built an image manipulation API to automate this process that takes a name input and generates a signed image. After succeeding, I made a GIPHY API that returned funny gif related to the subject of the bot.

Rituals ChatBot

The idea behind this case study was how a bot that delivers specific content behaves and how targeted users react to it. This bot reached to approximately 1000+ users. Throughout the process, I kept observing users interaction and enhancing service access for newly onboard users. I was successful to find out that more than 83% of users kept interacting with the bot compared to its first release. I learned a lot from this project and I was able to expand my Nodejs knowledge through this project by implementing multiple API as features were requested.
